HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element contains information used to determine how frequent the page appears in search results on Google or Bing.
  • The header tags should always be used in ascending order: from h1 to h6.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and the elements in the list, respectively.
  • The header tags range from h1 to h6, it is best to use them in ascending order to improve readability.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class attribute can be applied to multiple HTML elements to group them in a CSS rule.
  • A color can be designated for an element either semantically or by using hexadecimals.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls most recent commits from the remote main branch
  • git add -A: adds all new/edited files in repo to staging
  • git commit -m "": commits files in staging branch and provides a message that ideally gives a brief description of the changes made in the commit
  • git push origin main: pushes most recent commits to remote main branch
